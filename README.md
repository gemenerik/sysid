# sysid
Run rbf_network.py or ff_network.py for RBF networks or FF network, respectively. General network layout and setup is done in a class in nn_batches.py. The training is done by the definition in the same file. Uses optimizers and activation functions from utils folder. Plotting is done in plot_training.py and F16_PlotData.m that was customized from the one provided in the assignment.
